## ⚡ Comandos Rápidos

```bash

# Build
npm run build              # Build estático
npm run start              # Rodar build localmente

npm run lint               # Verificar erros de código

# Deploy
npm run export             # Export para HTML (GitHub Pages)
```

## 🆘 Problemas Comuns?

### Erro "Module not found"
```bash
rm -rf node_modules .next
npm install
npm run dev
```

### Site em branco
- Abra console (F12)
- Procure por erros em vermelho
- Verifique `data/portfolio.js`

### Não consegue fazer deploy
- Veja `DEPLOYMENT.md`
- Verifique se tem conta Vercel
- Verifique se código está no GitHub
